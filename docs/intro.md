> 구조부터 생각하며 시스템을 만듭니다.

웹 기반 **실시간 3D 스트리밍** 시스템과 백엔드 플랫폼을 설계하고 구현합니다.  
**WebRTC 기반 스트리밍 구조**, **서비스 아키텍처**, **런처·인프라 자동화**에 집중해왔습니다.

또한 **Unreal Engine**을 활용한 실시간 콘텐츠 실행 환경에도 익숙하며,  
엔진 단의 실행 구조와 서비스 단의 연결 흐름을 함께 고려하는 설계를 중요하게 생각합니다.

**완벽주의에 갇히기보다**, 운영 가능한 구조로 “계속 굴러가게” 만드는 쪽을 선택합니다.
